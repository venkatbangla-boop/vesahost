VESA Enterprise Inc - V15 Cinematic AI SEO Website

GitHub Pages-ready static website bundle for vesaent.com.
Upload the CONTENTS of this folder to the root of the GitHub repository.

Included:
- index.html
- css/style.css
- js/main.js
- assets/*
- manifest.webmanifest
- service-worker.js
- robots.txt
- sitemap.xml
- CNAME

V15 improvements:
- Cinematic premium buyer-facing layout
- Advanced CSS scroll reveal, counters, hover animation and route-line motion
- Stronger SEO metadata and JSON-LD schema
- EGENVA AI positioned as sourcing intelligence support
- Buyer RFQ-style enquiry section
- Responsive desktop and mobile design
- Official VESA and EGENVA logo assets from supplied materials

Deployment:
1. Unzip this package.
2. Upload all files and folders to the GitHub repository root.
3. Enable GitHub Pages.
4. Confirm CNAME points to vesaent.com.
5. Submit sitemap.xml in Google Search Console.
