VESA Enterprise Inc - Final GitHub Website V12

This version uses the approved generated visual set as clean standalone assets.

Included:
- index.html
- css/style.css
- js/main.js
- manifest.webmanifest
- service-worker.js
- robots.txt
- sitemap.xml
- assets folder

Visual system:
- Official VESA logo, EGENVA logo, and founder photo retained.
- Approved apparel sourcing visuals added as local image assets.
- No remote image dependencies.
- No cartoon visuals used on the webpage.
- No third-party employer logos.
- Print CSS added to reduce PDF rendering issues.

Deployment:
1. Upload all files to your GitHub repository root.
2. Enable GitHub Pages.
3. Point vesaent.com DNS to GitHub Pages.
4. Submit sitemap.xml in Google Search Console.
